  <Steps>
    <Step title="Install Highcharts">
      Install Highcharts using your preferred package manager:
      
      <CodeGroup>
      ```bash npm
      npm install highcharts
      ```
      ```bash yarn
      yarn add highcharts
      ```
      ```bash pnpm
      pnpm add highcharts
      ```
      </CodeGroup>
    </Step>

    <Step title="Import and Initialize">
      Import Highcharts in your JavaScript or TypeScript file:
      
      ```javascript
      import Highcharts from 'highcharts';
      ```
      
      Or use the CDN for quick prototyping:
      
      ```html
      <script src="https://code.highcharts.com/highcharts.js"></script>
      ```
    </Step>

    <Step title="Create Your First Chart">
      Add a container element and create a chart:
      
      ```html
      <div id="container"></div>
      ```
      
      ```javascript
      Highcharts.chart('container', {
        chart: {
          type: 'line'
        },
        title: {
          text: 'My First Chart'
        },
        xAxis: {
          categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May']
        },
        yAxis: {
          title: {
            text: 'Values'
          }
        },
        series: [{
          name: 'Sales',
          data: [29.9, 71.5, 106.4, 129.2, 144.0]
        }]
      });
      ```
      
      <Note>
        Highcharts automatically renders to the specified container. No additional setup required!
      </Note>
    </Step>

    <Step title="Customize and Explore">
      Explore the extensive configuration options to customize your chart. Check out our [chart types](/chart-types/line-area), [styling options](/features/styling-themes), and [API reference](/api/overview) for more details.
    </Step>
  </Steps>
